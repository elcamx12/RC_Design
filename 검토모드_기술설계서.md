> **작성일**: 2026-04-06

> **프로그램**: RC\_Design (Streamlit 웹앱)

> **버전**: v0.7.3(2026-04-06 17:00) 기준

> **목적**: 회의 자료 — 현재 구현 상태 + 향후 로드맵 방향성 검토용

## 1. 검토 모드 개요

### 1.1 목적

구조계산서(BeST.RC, BeST.Steel, MIDAS Gen)의 설계 결과를 **KDS 최신 기준**으로 독립 검증 및 수정이 가능하도록 한다.

### 1.2 핵심 원칙

| 구분         | 설명                                                     |
| ---------- | ------------------------------------------------------ |
| **입력(조건)** | 구조계산서 그대로 — Mu, Vu, Pu, 배근, 단면, fck, fy 등              |
| **계산(방법)** | KDS 최신 기준 — α1/β1(KDS 14 20 20), εcu, φ 전이구간, 전단 상세식 등 |
| **출력(결과)** | 구조계산서 배근/조건 기반 OK/NG 판정 + 수식 전개                        |

- 구조계산서 소프트웨어가 사용한 기준(구 기준 등)과 KDS 최신 기준이 다르기에, 구조계산서의 검토 결과와 본 프로그램의 검토 결과가 다를 수 있다.
- 사용자가 입력값(단면, 하중, 배근 등)을 수정하면 **수정된 조건으로 수동 재검토**가 가능하다 (자동 최적설계는 아님).

## 2. 전체 데이터 흐름

```
구조계산서 PDF
  ↓ (1) 위치 기반 자동 파싱 (parsers/pdf_parser.py)
부재별 데이터 추출 (단면, 재료, 부재력, 배근, 하중 등)
  ↓ (2) 입력 UI 표시 (ui/input.py)
사용자 확인 + 필요 시 값 수정 (수정된 값은 노란색 표시)
  ↓ (3) 독립 계산 (review/calculation_review.py)
KDS 기준 φMn, φVc, P-M 검토 등
  ↓ (4) 결과 출력 (ui/output.py)
요약 테이블 + 부재별 상세 (BeST/MIDAS 양식) + 시각화
```

## 3. 부재별 검토 상세

### 3.1 보 (Beam)

#### 3.1.1 파싱 항목

| 항목                               | 출처         | 수정 가능 |
| -------------------------------- | ---------- | :---: |
| 단면 (B × H, T형보: B\_top × H\_top) | 구조계산서      |   O   |
| fck, fy, fys                     | 구조계산서      |   O   |
| cover, Loc\_top, Loc\_bot        | 구조계산서 / 역산 |   O   |
| Mu(-), Mu(+), Vu (위치별)           | 구조계산서      |   O   |
| 배근 (TOP, BOT, Stirrup, Skin)     | 구조계산서      |   O   |
| Span                             | 구조계산서      |   O   |

#### 3.1.2 계산 수식

**등가응력블록 계수 (KDS 14 20 20 표 4.1-2)**

- fck ≤ 40 MPa: α1 = 0.80, β1 = 0.80
- 40 < fck ≤ 90: α1 = 0.80 - 0.004(fck - 40), β1 = 0.80 - 0.006(fck - 40)

**극한변형률 (KDS 14 20 20)**

- fck ≤ 40: εcu = 0.0033
- fck > 40: εcu = max(0.0033 - 0.0001 × (fck - 40)/10, 0.0026)

**휨 강도 φMn (KDS 14 20 20)**

복철근 모델 + Layer별 계산 (Skin 철근 포함):

- 각 철근 Layer별 변형률: εsi = εcu × (di - c) / c
- 각 Layer 응력: fsi = min(Es × εsi, fy) (압축/인장 구분)
- 힘 평형: α1 × fck × β1 × c × b = Σ(fsi × Asi)  →  c 수렴 (이분법)
- φMn = φ × Σ\[fsi × Asi × (di - a/2)]

단철근 모델 (슬래브 등 단순 케이스):

- a = (As × fy) / (α1 × fck × b)
- c = a / β1
- φMn = φ × As × fy × (d - a/2)

**강도감소계수 φ (KDS 14 20 10)**

- εt ≤ 0.002: φ = 0.65 (압축지배)
- 0.002 < εt < εt\_limit: φ = 0.65 + (εt - 0.002)/(εt\_limit - 0.002) × 0.20 (전이)
- εt ≥ εt\_limit: φ = 0.85 (인장지배)
- fy ≤ 400: εt\_limit = 0.005
- fy > 400: εt\_limit = 2.5 × εy

**전단 강도 — 상세식 적용 (KDS 14 20 22 §4.3.2)**

- Vc = (0.16√fck + 17.6ρw × Vud/Mu) × b × d
  - ρw = As\_tension / (b × d)
  - Vud/Mu ≤ 1.0
- 상한: Vc ≤ (5/18)√fck × b × d
- φVs = φ × Av × fy × d / s (스터럽 기여)
- 판정: φ(Vc + Vs) ≥ Vu, φ = 0.75

**T형보 지원**

- BeST에서 B\_top/H\_top 값이 있는 경우 T형보로 처리
- 압축블록이 플랜지 내에 있는지 확인 후 유효폭 적용

#### 3.1.3 판정 기준

| 검토 항목    | 판정                 |
| -------- | ------------------ |
| 휨 (부모멘트) | φMn ≥ Mu(-)        |
| 휨 (정모멘트) | φMn ≥ Mu(+)        |
| 전단       | φ(Vc + Vs) ≥ Vu    |
| 종합       | 모든 위치, 모든 항목 OK 판단 |

#### 3.2.2 SRC 합성기둥 지원

BeST.Steel 구조계산서에서 SRC(합성기둥)가 파싱되어 SRC 모드를 구현했다.

**현재 한계:**

- **각형강관(ㅁ형)만 지원** — 형식: `ㅁ-B×H×tv×th` (예: ㅁ-26x26x11x11)
- H형강, 원형강관 등은 미지원
- SRC φ = 0.90 (KBC 강구조 기준)
- 좌굴: AISC 기반 — Po/Pe 비율에 따라 Pn = Po × 0.658^(Po/Pe) 또는 0.877 × Pe

#### 3.2.3 계산 수식

**P-M 상관도 (KDS 41 20 22 §4.3)**

축력-모멘트 상호작용 곡선을 3개 기준점 + 50개 보간점으로 생성:

1. **순압축 (Pn,max)**
   - RC: Pn,max = α1 × fck × (Ag - As) + fy × As
   - φPn,max = 0.80 × φ × Pn,max (최대 축력 상한 80%)
   - SRC: Po = As,stl × fy,stl + As × fy + 0.85 × fck × Ac
2. **균형점 (Balanced)**
   - cb = εcu / (εcu + εy) × d
   - 중립축 cb에서 힘 평형으로 Pn,b / Mn,b 산정
3. **순휨 (Mn,o)**
   - Pn = 0이 되는 c를 이분법(60회)으로 탐색
   - φMn,o = φ\_tens × Mn,o
4. **P-M 곡선 보간**
   - 균형점 \~ 순휨 구간: c 값을 50등분하여 각 점의 (φPn, φMn) 산출
   - 각 c에서 εt 계산 → φ 전이구간 반영

**판정: 축력 + 단축 모멘트**

- 설계 (Pu, Mu)가 φP-M 곡선 내부에 있으면 OK

**이축 검토 — Bresler (KDS 41 20 22 §4.3.5)**

- X-X, Y-Y 각각 P-M 곡선에서 φMnx, φMny 산출
- 상호작용비: Rcom = Mux/φMnx + Muy/φMny ≤ 1.0

**세장비 검토**

- λ = k × Lu / r (k = 1.0 보수적 가정)
- λ ≤ 22: 단주 (모멘트 확대 불필요)
- λ > 22: 장주 → 모멘트 확대계수 δns 적용
  - δns = Cm / (1 - Pu/0.75Pc) ≥ 1.0 (Cm = 1.0)
  - Pc = π²EI / (kLu)²

**전단 (기둥)**

- 보 전단 상세식과 동일 (KDS 14 20 22 §4.3.2)

#### 3.2.4 판정 기준

| 검토 항목        | 판정                               |
| ------------ | -------------------------------- |
| P-M (X-X 방향) | (Pu, Mux)가 φP-M 곡선 이내            |
| P-M (Y-Y 방향) | (Pu, Muy)가 φP-M 곡선 이내            |
| 이축 (Bresler) | Rcom = Mux/φMnx + Muy/φMny ≤ 1.0 |
| 전단           | φ(Vc + Vs) ≥ Vu                  |
| 종합           | 모든 항목 OK                         |

### 3.3 슬래브 (1방향 슬래브)

#### 3.3.1 파싱 항목

| 항목                                 | 출처            |   수정 가능  |
| ---------------------------------- | ------------- | :------: |
| Slab Dim (Lx × Ly × H)             | 구조계산서         |     O    |
| fck, fy, cover                     | 구조계산서         |     O    |
| Edge Beams (UP/DN/LT/RT)           | 구조계산서         |     O    |
| Wd, Wl (사하중, 활하중)                  | 구조계산서         |     O    |
| Flexure Table (4행: Mu, ρ, Ast, 간격) | 구조계산서         | - (읽기전용) |
| 배근 선택 (combo + spacing, 4위치)       | 사용자 선택        |     O    |
| 지점 조건                              | 자동 감지 + 수동 변경 |     O    |

#### 3.3.2 계산 수식

**1방향 판별**

- β = Ly / Lx ≥ 2.0 → 1방향 슬래브 (단변이 주 하중 전달 방향)

**지점 조건과 모멘트 계수 (KDS 14 20 22 §4.3.2 근사해석법)**

현재 4가지 지점 조건을 지원하며, Mu = 계수 × Wu × Ln² 로 계산:

| 지점 조건 | 연속단 계수 | 중앙부 계수 | 비고    |
| ----- | :----: | :----: | ----- |
| 양단연속  |  -1/11 |  +1/16 | 내부 스팬 |
| 1단연속  |  -1/10 |  +1/11 | 외부 스팬 |
| 양단단순  |    0   |  +1/8  | 양단 핀  |
| 캔틸레버  |  -1/2  |    0   | 캔틸레버  |

> **확인 필요**: 현재 4가지 케이스를 만들어 두었으나, "양단 핀(양단단순)"으로만 둘지 스팬 길이/연속 조건에 따라 자동 조정하게 할지 교수님 확인 필요.

**계수하중**

- Wu = 1.2 × Wd + 1.6 × Wl (KDS 14 20 10 하중조합)

**순경간**

- Ln = Lx - (가장자리 보 폭)/2 (양쪽)
- 안전장치: Ln ≥ 0.8 × Lx

**최소 두께 (KDS 14 20 22 — 처짐 제어)**

- h\_min = Ln / ratio
  - 양단단순: ratio = 20
  - 1단연속: ratio = 24
  - 양단연속: ratio = 28
  - 캔틸레버: ratio = 10
- fy ≠ 400 보정: h\_min × (0.4 + fy/700)
- 판정: H ≥ h\_min

**독립 Mu 계산 (모멘트 계수법)**

- 단변: Mu = 계수 × Wu × (Ln/1000)² (kN·m/m)
- 장변: Mu = 0 (1방향 슬래브에서 장변은 주 하중 전달 방향이 아님. 온도/건조수축 철근 = 최소 배근만 배치)

> **확인 필요**: 장변 Mu = 0으로 두고 최소 철근비로 검토하는 현재 방식이 적절한지 교수님 확인 필요.

**독립 Mu와 BeST Mu 비교**

- 독립 계산 Mu와 구조계산서(BeST) Mu를 나란히 표시
- 차이율(%) = (독립 - BeST) / BeST × 100

**최소 철근비 (KDS 14 20 20 §4.5.1)**

- fy ≤ 400: ρ\_min = 0.002
- fy > 400: ρ\_min = 0.0018
- Ast\_min = ρ\_min × 1000 × H (mm²/m, 전체 두께 기준)

**배근 검토 (위치별 4개: Short Cont/Pos, Long Cont/Pos)**

- 사용자가 BeST Flexure Table에서 철근 조합(D10, D10+D13, D13 등)과 간격 선택
- 제공 As = 해당 조합의 단면적 / 간격 × 1000 (mm²/m)
- φMn 계산: 단철근 모델 (b = 1000mm/m)
- 판정: φMn ≥ Mu 이고 As ≥ Ast\_min

**전단 (KDS 14 20 22)**

- Vu = Wu × Ln / 2 (kN/m)
- Vc = (1/6) × √fck × b × d (N/m) → kN/m (간편식)
- φVc = 0.75 × Vc
- 판정: Vu ≤ φVc (슬래브는 스터럽 없음)

#### 3.3.3 판정 기준

| 검토 항목   | 판정                        |
| ------- | ------------------------- |
| 최소 두께   | H ≥ h\_min                |
| 휨 (4위치) | φMn ≥ Mu 이고 As ≥ Ast\_min |
| 전단 (X)  | Vu,x ≤ φVc,x              |
| 전단 (Y)  | Vu,y ≤ φVc,y              |
| 종합      | 모든 항목 OK                  |

## 4. 출력 구성

### 4.1 요약 테이블

모든 부재의 단면, 대표 배근, OK/NG 판정을 한 테이블로 표시.

### 4.2 부재별 상세 (탭 구조)

출력은 \*\*탭(슬래브 / 보 / 기둥)\*\*으로 구성되며, 각 부재를 탭으로 전환하며 확인할 수 있다.

**BeST 형식 출력 (HTML 테이블)**:

- Design Conditions (재료, 단면, 배근)
- 위치별 휨 검토 (수식 전개 + OK/NG)
- 전단 검토 (수식 전개 + OK/NG)
- 슬래브: Applied Loads, Min Thickness, Flexure Table (독립 Mu vs BeST Mu), Shear

**MIDAS 형식 출력**:

- 위치별(END-I/MID/END-J) 부모멘트/정모멘트/전단 개별 검토

## 6. 향후 로드맵

### 6.1 내진 배근 기본 설계 — IMF (KDS 14 20 80)

중간 모멘트 골조(Intermediate Moment Frame) 배근 상세 검토 추가.

**보 (IMF)**

- 소성힌지구간 띠철근 간격: ≤ min(d/4, 8db, 24dbt, 300mm)
- 이외 구간: ≤ d/2
- 모멘트 강도: 접합부면 Mn(+) ≥ 1/3 × Mn(-), 경간 내 Mn ≥ 1/5 × max(Mn)

**기둥 (IMF)**

- 소성힌지구간(L0) 길이: ≥ max(Lu/6, d, 450mm)
- 소성힌지 띠철근 간격 (SD400): ≤ min(d/2, 8db, 200mm)
- 이외 구간: 소성힌지 간격의 2배
- 첫 번째 띠철근: 접합면에서 간격의 1/2 이내

### 6.2 정착 길이 검토 — KDS 14 20 52

**표준갈고리 인장 정착 (KDS 14 20 52 §4.1.5)**

- lhb = 0.24 × db × fy × β / (λ × √fck)
- β: 에폭시 = 1.2, 무도막 = 1.0
- 최소: max(8db, 150mm)
- 보정계수 4가지 (피복, 90°갈고리+띠철근, 180°갈고리+띠철근, 초과배근)

**압축 정착 (KDS 14 20 52 §4.1.3)**

- ldb = 0.25 × db × fy / (λ × √fck)
- 최소: 200mm

> 정착 검토의 세부 범위(겹이음, 기둥 주근 정착 등)는 교수님 확인 후 결정.

### 6.3 확대머리 이형철근 — KDS 14 20 52 §4.1.6

보 철근이 기둥에 정착될 때 공간 확보가 어려운 경우 확대머리 도입을 고려.

**KDS 기준:**

- 접합부 정착: ldt = 0.22 × db × fy / (ψ × √fck)
  - ψ = 0.6 + 0.3(cso/db) + 0.35(Ktr/db) ≤ 1.375
- 최소: max(8db, 150mm)
- 적용 조건: 순피복 ≥ 1.35db, 순간격 ≥ 2db, 확대머리 뒷면 횡철근 50mm 이내

> 공간 부족 시 대안으로 도입. 기본 옵션 포함 여부는 추후 결정.

### 6.4 슬래브 3D 배근 시각화

- 3D 골조 배근도에 슬래브 배근 추가 (보+기둥과 함께 표시)
- 부재별 끄기/켜기 옵션 (슬래브, 보, 기둥 개별 토글)
