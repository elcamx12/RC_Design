# 🏗️ [Project Documentation] RC 구조 설계 자동화 시스템

이 문서는 현재 구현된 **RC 보(Beam), 기둥(Column), 슬래브(Slab) 설계 자동화 시스템**의 계산 로직, 적용 기준(KDS 41 20 00), 그리고 프로세스 흐름을 상세히 기술합니다.

---

## 🏛️ 1. 시스템 개요 및 구조

이 시스템은 `Streamlit` 기반의 웹 애플리케이션으로, 사용자가 입력한 기하/하중/재료 조건을 바탕으로 KDS 41 20 00 건축물 콘크리트구조 설계기준을 만족하는 최적의 단면과 철근 배근을 자동으로 산출합니다. 보(휨/전단/균열/정착/처짐/IMF), 기둥(P-M/세장비/접합부/IMF), 슬래브(1방향 휨/전단/처짐/배력근)를 통합 설계합니다.

### 📁 핵심 파일 구조
*   **`app.py`**: 애플리케이션의 진입점(Entry Point). `ui`와 `calculation_manager`를 연결합니다.
*   **`ui/`**: Streamlit UI 폴더 (`__init__.py` re-export)
    *   `input.py`: 입력 섹션 UI (`render_input_section`)
    *   `output.py`: 출력 섹션 UI (`render_output_section`)
    *   `pdf_import.py`: PDF 파싱 UI (`_pdf_import_section`)
*   **`calculation_manager.py`**: 전체 계산 흐름을 제어하는 중앙 관리자. 입력값을 받아 각 엔진(`beam_engine`, `column_engine`)을 호출하고 결과를 취합합니다.
*   **`beam_engine.py`**: **보(Beam)**의 해석 및 설계 로직(휨, 전단, 배근)을 담당하는 `BeamAnalyzer` 클래스가 포함되어 있습니다.
*   **`column_engine.py`**: **기둥(Column)**의 해석 및 설계 로직(P-M 상관도, 띠철근)을 담당하는 `ColumnAnalyzer` 클래스가 포함되어 있습니다.
*   **`visualization/`**: Plotly 시각화 폴더 (`__init__.py` re-export)
    *   `beam.py`: 보 시각화 (단면도, 측면도, SFD/BMD)
    *   `column.py`: 기둥 시각화 (단면도, 측면도, P-M)
    *   `slab.py`: 슬래브 단면도
    *   `frame_3d.py`: 3D 골조 부재력도/배근도
    *   `helpers.py`: 공유 3D 유틸 (create_box 등)
*   **`slab_engine.py`**: **슬래브(Slab)**의 1방향 구조설계(휨, 전단, 처짐, 배력근)를 담당하는 `SlabAnalyzer` 클래스가 포함되어 있습니다.
*   **`parsers/`**: 구조계산서 PDF 분석 모듈 (`pdf_parser.py`, `ai_pdf_parser.py`, `ui_pdf_viewer.py`).
*   **`launcher/`**: 앱 실행 관련 파일 (`index.html` 랜딩 페이지, `start.py` Python 런처).
*   **`RC_Design.bat`**: 앱 실행 런처 (더블클릭 → Streamlit 서버 시작 + HTML 랜딩 페이지 열기).

## 🌉 2. 보(Beam) 설계 로직 (`beam_engine.py`)

### 2.1 하중 산정 및 해석
1.  **단면 가정**:
    *   **춤(h)**: $L/28$ (최적화 시작점, 처짐 반복 설계가 실제 필요 깊이를 자동 산출). KDS L/21 참조값은 비교 표시용으로 유지. 최소 150mm, 50mm 단위 올림.
    *   **폭(b)**: $\min(\text{round\_up\_to\_50}(h \times 0.5),\ h)$ — 고정 최솟값 없음 (구 300mm 하한 제거). 배근 불가(2단 폴백) 시 $b += 50\text{mm}$ 반복. $b > h$ 시 $h$ 증가로 전환.
    *   **주근 탐색**: D13 ~ D32 (D13 추가됨, 2026-03-10)
    *   **강도감소계수($\phi$)**: $\varepsilon_t$에 따른 전이구간 선형 보간 수렴 루프 적용 (I1, 2026-03-10)
2.  **슬래브 하중 분배 (1방향 슬래브 가정)**:
    *   보의 길이 비율에 따라 긴 변 방향 보가 하중을 부담합니다.
    *   하중 분담 폭($S_{slab}$)은 짧은 변 길이의 절반($L_{short}/2$)으로 설정됩니다.
3.  **하중 조합 (KDS 41 10 15)**:
    *   $w_u = 1.2 D + 1.6 L$
    *   $D$ (고정하중): 슬래브 자중 + 보 자중 + 추가 마감하중($DL_{area}$)
    *   $L$ (활하중): 용도별 활하중($LL_{area}$)
4.  **부재력 계산 (고정단 보, 강접합)**:
    *   지점부 부모멘트: $M_{neg} = w_u L^2 / 12$
    *   중앙부 정모멘트: $M_{pos} = w_u L^2 / 24$
    *   최대 전단력: $V_u = w_u L / 2$
    *   BMD: $M(x) = -M_{neg} + V_{max} \cdot x - w_u \cdot x^2 / 2$
    *   상부근(M_neg 저항) + 하부근(M_pos 저항) 이중 배근 설계

### 2.2 휨 설계 (Flexural Design)
*   **적용 기준**: 강도설계법 (KDS 41 20 20)
*   **강도감소계수($\phi$)**: 인장지배단면 0.85, 전이구간 선형 보간, 압축지배 0.65
*   **등가응력블록 깊이 계수($\beta_1$)**: 콘크리트 강도($f_{ck}$)에 따라 0.85 ~ 0.65 가변 적용.
*   **필요 철근량($A_{st}$) 산정**:
    *   $M_u = \phi M_n$ 식을 이용하여 필요 철근비($\rho_{req}$)를 2차 방정식으로 역산.
    *   **2-패스 유효깊이(d) 추정**: 1패스(D25 가정) → As 추정 → 최적 철근직경 선택 → 2패스 d_c 정밀화
*   **철근비 검토**:
    *   **최소 철근비($\rho_{min}$)**: $\max(0.25\sqrt{f_{ck}}/f_y, 1.4/f_y)$ 이상 확보.
    *   **최대 철근비(인장지배)**: 최외단 인장철근 순인장변형률($\epsilon_t$) $\ge 0.005$ 만족 여부 확인 (압축지배 시 경고).

### 2.3 전단 설계 (Shear Design)
*   **적용 기준**: KDS 41 20 22
*   **강도감소계수($\phi$)**: **0.75**
*   **전단강도 분담**:
    *   콘크리트($V_c$): $\frac{1}{6} \sqrt{f_{ck}} b_w d$
    *   전단철근($V_s$): $V_u/\phi - V_c$
*   **늑근(Stirrup) 간격 산정**:
    *   이론적 간격: $s_{req} = A_v f_{yt} d / V_s$
    *   최소 철근량 간격: $s_{max,Av}$ (KDS 4.3.2)
    *   기하학적 최대 간격: $V_s$ 크기에 따라 $d/2$ (또는 600mm) 혹은 $d/4$ (또는 300mm).
    *   **최종 간격**: 위 3가지 중 최솟값을 선택하고 **50mm 단위로 내림(Round Down)** 처리 (최소 100mm).
*   **스터럽 구간 설계** (`calculate_shear_zones`): 보를 n_zones 구간으로 분할, 구간별 Vu에 맞는 간격 산정. 대칭 미러링 적용.

### 2.4 철근 상세 (Rebar Detailing)
*   **주철근 선정**: D13 ~ D32 규격 중 소요 면적을 만족하는 최소 가닥수 탐색.
*   **배근 적합성 검토**:
    *   보 폭($b$) 내에 철근이 1단으로 배치 가능한지 검토 (피복두께, 늑근직경, 굵은골재 최대치수 고려한 최소 순간격 $S_{min}$ 적용).
    *   1단 배근 불가능 시 **2단 배근**으로 자동 전환 (경고 메시지 출력).
*   **상부/하부 직경 통일**: 상부근과 하부근의 직경이 다를 경우, 큰 쪽으로 통일하여 시공성 확보.

### 2.5 균열 검토 (Crack Control) — KDS 41 20 30 4.2
*   **적용 기준**: KDS 41 20 30 4.2 — 간접적 균열폭 제한 (철근 간격 기반)
*   **철근 응력**: $f_s = M_a / (A_s \cdot j_d)$ — 사용하중 모멘트 기준 (계수하중의 60% 근사)
*   **최대 간격 한도**:
    *   $s_1 = 380 \cdot (k_{cr} / f_s) - 2.5 \cdot c_c$
    *   $s_2 = 300 \cdot (k_{cr} / f_s)$
    *   $s_{max} = \min(s_1, s_2)$
*   $k_{cr}$: 노출조건 계수 (건조환경 280, 습윤환경 210)
*   판정: 실제 철근 간격 $s_{actual} \le s_{max}$ → OK

### 2.6 정착길이 (Development Length) — KDS 41 20 50
*   **기본 정착길이**: $\ell_{db} = 0.9 \cdot d_b \cdot f_y / \sqrt{f_{ck}}$
*   **보정계수**: 상부철근 α=1.3, 에폭시 β=1.0, 경량콘크리트 γ=1.0
*   **cb, Ktr 보정**: $(c_b + K_{tr}) / d_b \le 2.5$ — 현재 Ktr=0 (보수적)
*   **감소계수**: $\ell_d = \ell_{db} \times (A_{s,req} / A_{s,prov})$
*   **겹이음**: B급 일률 적용 $\ell_s = 1.3 \cdot \ell_d$

### 2.7 처짐 검토 (Deflection) — KDS 41 20 30 4.3
*   **Branson 유효단면2차모멘트**: $I_e = I_{cr} + (I_g - I_{cr})(M_{cr}/M_a)^3$
*   **가중평균**: $I_{e,avg} = 0.70 \cdot I_{e,mid} + 0.30 \cdot I_{e,sup}$ (양단 고정 대칭)
*   **즉시처짐**: $\delta_i = w L^4 / (384 E_c I_e)$
*   **장기처짐**: $\delta_{lt} = \lambda_\Delta \cdot \delta_{i,sus}$ — $\lambda_\Delta = \xi / (1 + 50\rho')$, $\xi$ = 경과시간 계수
*   **허용처짐**: $L/480$ (비구조부재 손상 고려) 또는 $L/240$ (일반)

### 2.8 IMF 보 내진 상세 (Seismic Detailing) — KDS 41 17 4.4
*   **소성힌지 구간**: $\ell_{ph} = 2h$ (부재 양단)
*   **구간 내 스터럽 최대 간격**: $\min(d/4,\ 8d_b,\ 24d_{stirrup},\ 300\text{mm})$
*   **하부근 비율**: 소성힌지 구간에서 하부근 ≥ 상부근의 1/2 이상

## 🏛️ 3. 기둥(Column) 설계 로직 (`column_engine.py`)

### 3.0 기둥 단면 결정
*   **초기 단면**: $c = \text{round\_up\_to\_50}(\max(b_x, b_y) + 100)$ — 보 폭 기반, 고정 최솟값 없음 (400mm 제거됨, 2026-03-10)
*   **세장비 수렴 루프** (`calculation_manager.py`): λ>100 또는 좌굴 불안정 → c += 50mm 재시작
*   **P-M 설계 루프** (`column_engine.py`): 철근비 > 8% → c += 50mm → 세장비 재검토

### 3.1 하중조합 (KDS 41 10 15)
*   다중 하중조합 검토 후 지배조합 선택:
    *   1.2D + 1.6L
    *   1.4D
    *   1.2D + 1.0L + 1.0E (지진 활성 시)
    *   0.9D + 1.0E (지진 활성 시)
*   축력 분해: Pu_add(상부층) + V_beam(보 전단반력) + P_self(기둥 자중) → Pu_total
*   모멘트 분해: Mux_add + M_neg(보 단부 모멘트) → 방향별 합산 후 SRSS

### 3.2 P-M 상관도 기반 설계 (Interaction Diagram)
단순 중심축하중 설계가 아닌, 휨모멘트($M_u$)와 축하중($P_u$)을 동시에 고려하는 **P-M 상관도법**을 적용합니다.

1.  **최소 편심 보정**:
    *   실제 편심 $e = M_u / P_u$ 계산.
    *   최소 편심 $e_{min} = 15 + 0.03h$ (KDS 기준).
    *   만약 $e < e_{min}$이면, 설계 모멘트를 $M_{u,design} = P_u \times e_{min}$으로 할증하여 적용.
2.  **P-M 포락선(Envelope) 구성**:
    *   철근비 1% ~ 8% 범위 내에서 반복 시산(Iteration).
    *   **점 A (순수 압축)**: $\phi P_{n,max} = 0.80 \phi [0.85 f_{ck} (A_g - A_{st}) + f_y A_{st}]$ ($\phi=0.65$)
    *   **점 B (균형 파괴)**: 인장철근이 항복함과 동시에 압축연단 콘크리트가 파괴되는 지점 ($\epsilon_c=0.003, \epsilon_s=\epsilon_y$).
    *   **점 C (순수 휨)**: 축력이 0일 때의 휨강도 $\phi M_n$.
    *   **이축 휨**: Bresler Method 적용 — $1/P_r = 1/P_{rx} + 1/P_{ry} - 1/P_0$
3.  **안전성 검토**:
    *   설계 하중점 $(M_u, P_u)$가 위 3점을 잇는 포락선 내부에 존재하는지 판별.
    *   NG일 경우 철근 가닥수를 증가시키거나, 철근비가 8%를 초과하면 기둥 단면을 키워 재계산.

### 3.3 세장비 검토 (Slenderness) — KDS 41 20 40
*   **세장비**: $\lambda = k \cdot l_u / r$ — 현재 k=1.0 (보수적)
*   **단주 판정**: $\lambda \le 22$ → 세장 효과 무시
*   **장주 모멘트 증폭**: $M_c = \delta_{ns} \cdot M_u$ — $\delta_{ns} = C_m / (1 - P_u / P_c) \ge 1.0$
*   **좌굴하중**: $P_c = \pi^2 E I / (k l_u)^2$ — EI = 0.4·Ec·Ig / (1+βd)

### 3.4 띠철근(Tie Rebar) 상세
*   **직경 선정**: 주철근 D32 이하 → **D10**, D35 이상 → **D13**.
*   **간격 산정**: 다음 3가지 값 중 **최솟값**을 선택하고 50mm 단위로 내림.
    1.  주철근 지름의 16배 ($16 d_b$)
    2.  띠철근 지름의 48배 ($48 d_t$)
    3.  기둥 단면의 최소 치수

### 3.5 접합부 전단 검토 (Joint Shear) — KDS 41 17
*   $V_{jh} = A_s \cdot f_y - V_{col}$ (보 상부근 인장력 - 기둥 전단력)
*   $\phi V_n = \phi \cdot \gamma \cdot \sqrt{f_{ck}} \cdot A_j$ — γ는 접합부 유형에 따라 결정
*   X, Y 방향 각각 독립 검토

### 3.6 IMF 기둥 내진 상세 — KDS 41 17
*   **구속 구간**: $l_o = \max(c,\ l_u/6,\ 450\text{mm})$ (기둥 양단)
*   **구속 구간 내 최대 간격**: $\min(c/2,\ 8d_b,\ 24d_{tie},\ 300\text{mm})$
*   **강주약보(Strong-Column-Weak-Beam)**: $\sum M_{n,col} \ge 1.2 \sum M_{n,beam}$

## 🧱 4. 슬래브(Slab) 설계 로직 (`slab_engine.py`)

### 4.1 모델링
*   **1방향 슬래브**: 단위폭 1m 스트립 모델 (양단 고정)
*   **두께 결정**: $t = L_{short}/20$ (50mm 단위 올림, 최소 150mm). 사용자 직접 지정도 가능.

### 4.2 하중 산정
*   $w_u = 1.2 \cdot (t \cdot 24 + DL_{area}) + 1.6 \cdot LL_{area}$ (kN/m, 1m 스트립)
*   부재력: $M_{neg} = w_u L^2/12$, $M_{pos} = w_u L^2/24$, $V_{max} = w_u L/2$

### 4.3 휨 설계
*   보와 동일한 강도설계법, 단 b=1000mm (1m 스트립)
*   **최소 철근비**: 수축·온도 철근비 $\rho_{min} = 0.002$ (fy=400), $0.0018$ (fy=500)
*   결과는 간격(spacing) 기반: D10@200 형식

### 4.4 전단 검토
*   슬래브는 일반적으로 전단철근 불필요: $V_u \le \phi V_c$ 확인
*   $V_c = \frac{1}{6} \sqrt{f_{ck}} \cdot b_w \cdot d$

### 4.5 처짐 검토
*   보와 동일한 Branson 방법
*   지점부(Icr_sup) / 중앙부(Icr_mid) 분리 계산 → 가중평균
*   경계조건(양단고정/일단연속 등) 반영

### 4.6 배력근 (Distribution Rebar)
*   주철근 방향 직교 배치: $\rho_{dist} = \max(0.002, 220/f_y)$ (KDS 41 20 20)
*   결과: 간격 기반 (D10@300 등)

### 4.7 균열 검토 / 정착길이
*   보와 동일한 로직 적용 (간격 기반 s_max 검토, 정착길이 산정)
*   피복두께 20mm (슬래브 기본값)

## 📄 구조계산서 PDF 연동 계획 (MIDAS/BeST.RC → RC_Design)

### 입력값 (AI가 구조계산서에서 읽어오는 값)

| 부재 | 읽어올 값 | 비고 |
|------|-----------|------|
| **보** | Mu(END-I, MID, END-J), Vu(END-I, END-J), 단면(b, h) | 포락값(ENV) 또는 조합별 최대값 |
| **기둥** | Pu, Mux, Muy, Vu, 단면(bx, by) | 하중조합별 값 |
| **슬래브** | Wd(고정하중), Wl(활하중), 단면 두께, 경간 | 면하중은 하중표에서 추출 |

### 설계 흐름 (부재력 직접 입력 → 단면 검토)

```
구조계산서 PDF
  ↓ AI 파싱
Mu, Vu, Pu (계수하중 조합값)
  ↓ 엔진 입력
단면설계: 휨(As), 전단(스터럽), 균열, 정착길이, 철근배치
  ↓
결과: OK/NG 판정 + 배근 상세
```

*   기존 모드(하중 입력 → 부재력 산정 → 단면설계)와 별도로 **"검토 모드"** 추가
*   검토 모드에서는 단면이 이미 정해져 있으므로 **수렴 루프(단면 키우기)를 사용하지 않음**
*   보의 경우 상부근 = max(|M_neg_i|, |M_neg_j|), 하부근 = M_pos_mid 로 설계 (양단 동일 배근, 실무 표준)

### 가능한 검토 항목

| 항목 | 가능 여부 | 설명 |
|------|:---------:|------|
| 휨 설계 (As 산정) | O | Mu 직접 사용 |
| 전단 설계 (스터럽) | O | Vu 직접 사용 |
| 균열 검토 | O | As 확정 후 fs 계산 |
| 정착길이 | O | As, fck, fy로 산정 |
| 철근 배치 (간격/피복) | O | 단면 b, 철근 개수로 검증 |
| **처짐** | **X** | D/L 분리 모멘트 필요 — 구조계산서에 포락값만 있어 불가. MIDAS 처짐 결과를 직접 입력받아 한도 비교(OK/NG)만 수행 |
| **수렴 루프 (단면 자동 조정)** | **X** | 부재력이 단면에 종속 → MIDAS 재해석 필요. 검토 모드에서는 판정만 |

### MIDAS vs BeST.RC 병용 이유 (참고)

구조계산서에서 동일 부재를 MIDAS와 BeST.RC 양쪽으로 검토하는 경우가 있음:
*   **MIDAS Gen**: 골조 해석 → 부재력(Mu, Vu, Pu) 산출 + 부재 설계 검토
*   **BeST.RC**: 독립 부재 단면설계 프로그램 → MIDAS 결과의 교차 검증(cross-check) 용도
*   두 프로그램의 결과가 일치하는지 확인하여 설계의 신뢰성을 높이는 실무 관행

### ⚠️ Ec 탄성계수 Δf 보정 (보류)
- **현재**: `Ec = 8500·(fck+4)^(1/3)` — Δf=4 고정 (fck≤40에서 정확)
- **정확한 공식**: `Ec = 8500·(fck+Δf)^(1/3)` — Δf는 fck에 따라 4~6 MPa 보간
  - fck ≤ 40: Δf=4 (현재와 동일), fck ≥ 60: Δf=6, 40~60: 선형 보간
- **상태**: 보류 (일반 건축물 fck=21~40에서 영향 없음, 고강도 지원 시 적용)
- **적용 위치**: `beam_engine.py`, `column_engine.py`
