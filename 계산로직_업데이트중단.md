# RC_Design 계산 로직 문서

> **목적**: 본 문서는 RC_Design 시스템의 모든 구조 계산 과정을 실행 순서대로 기술합니다.
> 실무 엔지니어가 처음부터 끝까지 계산의 무결성을 점검할 수 있도록 매 단계의 입력·식·출력·판정 기준을 명시하였으며, 각 식이 근거하는 KDS 조항을 병기합니다.

---

## 0. 용어 및 단위 규약

| 기호 | 의미 | 단위 |
|------|------|------|
| L_x, L_y | X/Y 방향 보 경간 | mm (입력) → m (내부) |
| h_column | 기둥 층고 (비지지 길이) | mm |
| DL_area | 추가 마감 고정하중 (슬래브 자중 제외) | kN/m² |
| LL_area | 활하중 | kN/m² |
| fc_k (fck) | 콘크리트 설계기준 압축강도 | MPa |
| fy | 철근 항복강도 | MPa |
| Pu | 계수 축력 | kN |
| Mu, Mux, Muy | 계수 휨모멘트 | kN·m |
| d | 유효깊이 | mm |
| d_c | 인장연단 ~ 철근 중심 거리 | mm |
| As | 철근 단면적 | mm² |
| φ (phi) | 강도감소계수 | 무차원 |
| ε_cu | 콘크리트 극한변형률 = 0.003 | 무차원 |
| β₁ (beta1) | 등가직사각형 응력블록 깊이 계수 | 무차원 |

> **내부 단위**: 경간 L은 m, 하중은 kN/m, 모멘트는 kN·m, 단면치수는 mm.
> 힘 평형식 계산 시 N·mm 단위로 변환 후 수행합니다.

## 2. 슬래브 두께 산정

> **KDS 41 20 30 Table 4.3-1** — 양단 고정 1방향 슬래브 최소두께

### 2.1 최소두께 산정

```
L_short = min(L_x, L_y)                   ← mm
t_slab_raw = L_short / 20.0               ← mm (양단 고정 최소 깊이)
t_slab = max(⌈t_slab_raw⌉₅₀, 150)        ← mm (50mm 단위 올림, 최소 150mm)
```

> `⌈x⌉₅₀` = `ceil(x / 50) × 50`

### 2.2 슬래브 하중 분배

```
aspect_ratio = L_long / L_short   (L_long = max(L_x, L_y))
```

**1방향 슬래브 (aspect_ratio ≥ 2.0)**:
- 단변 방향 보만 하중 부담
- `S_slab_short방향보 = L_short / 2`, 장변 방향 보 분담폭 = 0

**2방향 슬래브 (aspect_ratio < 2.0)**:
- 등가등분포하중 기반 분담폭 (KDS 41 20 30)
- `r = L_short / L_long` (≤ 1.0)
- 장변보 분담폭: `S_long = L_short × (3 - r²) / 6`  (사다리꼴 분배 등가)
- 단변보 분담폭: `S_short = L_short / 3`  (삼각형 분배 등가)

## 3. 1방향 슬래브 구조설계

> **모델**: 1m 폭 스트립, 양단 고정보

### 3.1 하중 산정

> **KDS 41 10 15** — 하중조합 1.2D + 1.6L

```
w_slab_self = (t_slab / 1000) × 24.0                    [kN/m²]  ← 콘크리트 자중
w_DL = w_slab_self + DL_area                             [kN/m]   ← 1m 스트립
w_LL = LL_area                                           [kN/m]
w_u  = 1.2 × w_DL + 1.6 × w_LL                          [kN/m]
```

### 3.2 부재력 (고정단 보)

```
M_neg = w_u × L² / 12     [kN·m]  ← 지점부 부모멘트 (상부 인장)
M_pos = w_u × L² / 24     [kN·m]  ← 중앙부 정모멘트 (하부 인장)
V_max = w_u × L / 2       [kN]    ← 지점 최대 전단력
```
> L = L_short (m 단위)

### 3.3 휨 설계

> **KDS 41 20 20 4.2** — 강도설계법

**유효깊이**:
```
cover = 20mm  (KDS 41 20 50 — 옥내 슬래브)
d_c = cover + d_b/2       ← 스터럽 없음 (슬래브)
d = t_slab - d_c           [mm]
```

**β₁ (KDS 41 20 20 4.1.1 (4))**:
```
fck ≤ 28 MPa : β₁ = 0.85
fck > 28 MPa : β₁ = max(0.65, 0.85 - 0.05/7 × (fck - 28))
```

**φ-ε_t 수렴 루프** (최대 10회 반복):

```
1. Mn = Mu / φ                                           [N·mm]
2. Rn = Mn / (b × d²)                                    [MPa]
3. discriminant = 1 - 2Rn / (0.85 × fck)
   ← discriminant < 0 → 단면 부족 (설계 불가)
4. ρ = (0.85 × fck / fy) × (1 - √discriminant)
5. As = ρ × b × d → a = As·fy / (0.85·fck·b) → c = a/β₁
6. ε_t = ε_cu × (d - c) / c                               [KDS 41 20 20 4.2.2]
7. φ 갱신:
   ε_t ≥ 0.005 → φ = 0.85  (인장지배)
   ε_t ≤ 0.002 → φ = 0.65  (압축지배)
   그 외     → φ = 0.65 + 0.20 × (ε_t - 0.002) / 0.003  (전이구간)
8. |φ_new - φ_old| < 1e-4 → 수렴 완료
```

**최소 철근비** (KDS 41 20 20):
```
fy ≥ 400 : ρ_min_temp = 0.0018
fy ≤ 300 : ρ_min_temp = 0.0020
그 외    : ρ_min_temp = 0.0020 - 0.0002 × (fy - 300) / 100

ρ_min_flex = max(0.25√fck / fy, 1.4 / fy)
ρ_min = max(ρ_min_temp, ρ_min_flex)

ρ < ρ_min → ρ = ρ_min (상향 조정)
```

**최종**: `As = ρ × b × d` [mm²/m]

### 3.4 배근 상세

> 간격 기반 배근: "D10@200" 형태

```
후보: D10, D13, D16 (순서대로 탐색)
s_max = min(3 × t_slab, 450mm)          [KDS 41 20 20]

각 규격 d_b에 대해:
  s_req = 1000 × A_b / As_req           ← 소요 간격
  s = ⌊s_req / 25⌋ × 25                 ← 25mm 내림
  s = max(s, 100mm)
  s ≤ s_max 이고 As_prov ≥ As_req → 채택
```

### 3.5 배력근 (수축·온도 철근)

> **KDS 41 20 20** — 경간 직각 방향

```
ρ_dist: fy≥400 → 0.0018 / fy≤300 → 0.0020 / 그 외 보간
d_dist = t_slab - 20 - 9.53/2           ← D10 기준 유효깊이 근사
As_dist = ρ_dist × 1000 × d_dist        [mm²/m]
```
→ 동일 배근 탐색 알고리즘 적용

### 3.6 전단 검토

> **KDS 41 20 22 4.3.1** — 위험단면 (지점면에서 d 위치)

```
d = t_slab - cover - d_b/2                                [mm]
V_at_d = V_max - w_u × (d/1000)                           [kN]  ← 위험단면 전단력

## 4. 보 설계 (X/Y 방향)

> **수렴 루프**: h 증가(처짐) → b 증가(배근) → 반복 (최대 10회)

### 4.1 설계 흐름

```
1. 초기 단면: h = ⌈L/28⌉₅₀ (최적화 시작점), b = ⌈h×0.5⌉₅₀
2. 하중 산정 → 부재력
3. 휨 설계 (상부근 M_neg, 하부근 M_pos)
4. 배근 상세 (1단 배근 가능 여부)
   ├ 1단 불가 → b += 50mm (b ≤ h 범위 내), 3번으로
   └ b > h → h += 50mm, b 리셋, 3번으로
5. 처짐 검토
   ├ OK → 전단 설계로 진행
   └ NG → h += 50mm, b 리셋, 3번으로
6. 전단 설계 (처짐 확정 후 1회)
```

### 4.2 하중 산정

> **KDS 41 10 15** — 1.2D + 1.6L

```
w_slab_self = (t_slab / 1000) × 24.0                    [kN/m²]
w_beam_self = (b/1000) × ((h - t_slab)/1000) × 24.0     [kN/m]  ← 보 돌출부 자중
w_DL = (w_slab_self + DL_area) × S_slab + w_beam_self   [kN/m]  ← 비계수 고정하중
w_LL = LL_area × S_slab                                  [kN/m]  ← 비계수 활하중
w_u  = 1.2 × w_DL + 1.6 × w_LL                          [kN/m]  ← 계수하중
```

> S_slab: §2.2에서 결정된 해당 방향 분담폭 (m 단위)
> 보 자중 산정 시 b×t_slab 부분은 슬래브 자중에 이미 포함되므로 제외

### 4.3 부재력 (양단 고정보)

```
M_neg = w_u × L² / 12     [kN·m]    ← 지점부 부모멘트
M_pos = w_u × L² / 24     [kN·m]    ← 중앙부 정모멘트
V_max = w_u × L / 2       [kN]      ← 지점 최대 전단력
```

### 4.4 휨 설계

> **KDS 41 20 20 4.2** — 슬래브(§3.3)와 동일한 φ-ε_t 수렴 루프

**유효깊이 (2패스 추정)**:
```
1패스: D25 가정 → d_c_init = 40 + 10 + 25.4/2 = 62.7mm → d_init
       → ρ 추정 → As 추정 → 최적 철근경 d_b 추정
2패스: d_c = 40(cover) + 10(stirrup) + d_b/2 → d = h - d_c
```

> cover = 40mm (KDS 41 20 52 — 보), 늑근 D10 = 10mm

**최소 철근비** (KDS 41 20 20):
```
ρ_min = max(0.25√fck / fy, 1.4 / fy)
```

**최대 철근비 (ε_t ≥ 0.004 기준)**:
```
ρ_max = 0.85 × β₁ × fck / fy × 0.003 / (0.003 + 0.004)
ε_t < 0.004 → KDS 최대 철근비 초과 경고
```

### 4.5 배근 상세

> 가닥수 기반: "5-D25" 형태

```
후보: D13 → D16 → D19 → D22 → D25 → D29 → D32

1단 배근 검토:
  b_net = b - 2×cover - 2×d_stir
  S_min = max(4/3 × 25mm, d_b)             [KDS 41 20 20 — 최소 순간격]
  n = ⌈As_req / A_b⌉  (최소 2가닥)
  req_width = n×d_b + (n-1)×S_min
  req_width ≤ b_net → 채택

2단 배근 폴백:
  D25 고정, 가닥수 산정 → layer = 2
```

**직경 통일**: 상·하부근 직경이 다를 경우, 큰 직경으로 양쪽 모두 재배근

### 4.6 전단 설계

> **KDS 41 20 22** — 전단 강도 설계

#### 4.6.1 위험단면 전단력 (KDS 41 20 22 4.3.1)

```
d_m = d / 1000                                           [m]
V_at_d = V_max - w_u × d_m                               [kN]
```
> 위험단면: 지점면에서 d 떨어진 곳. V_at_d < V_max → 전단설계 경제적

#### 4.6.2 전단 강도 계산

```
φ = 0.75                                                  [전단 강도감소계수]
Vn = Vu / φ                                               [N]

Vc = (1/6) × λ × √fck × b × d                            [N]  ← 콘크리트 전단강도
Vs = Vn - Vc                                              [N]  ← 전단철근 부담분

[단면 부족 검토]
Vs_max = (2/3) × √fck × b × d                            [N]
Vs > Vs_max → "단면 파괴" — 단면 크기 증가 필요

[늑근 간격 산정]  A_v = 2 × 71.33 = 142.66 mm² (D10 2-legs)
s_req = A_v × fy_t × d / Vs                              [mm]  ← 요구 간격
s_max_Av = A_v × fy_t / max(0.0625√fck × b, 0.35 × b)   [mm]  ← 최소 전단철근량 간격

[기하학적 최대 간격]  (KDS 41 20 22)
Vs ≤ (1/3)√fck×b×d → s_max_geom = min(d/2, 600mm)
Vs > (1/3)√fck×b×d → s_max_geom = min(d/4, 300mm)

s_final = ⌊min(s_req, s_max_Av, s_max_geom) / 50⌋ × 50
s_final = max(s_final, 100mm)
```

#### 4.6.3 구간별 늑근 설계

```
반경간을 n_zones 등분 (위험단면 ~ 중앙)
각 구간의 설계 전단력 = 구간 시작점(지점 쪽) Vu 사용 (보수적)

구간 i:
  x_s = x_start + i × L_eff / n_zones
  Vu = max(|V_max - w_u × x_s|, |V_max - w_u × x_e|)
  → §4.6.2 동일 절차로 s 산정
```

### 4.7 처짐 검토

> **KDS 41 20 30 4.3** — Branson 유효 단면2차모멘트

#### 4.7.1 재료 특성

```
Ec = 8500 × (fck + 4)^(1/3)                              [MPa]  ← KDS 41 20 00
fr = 0.63 × √fck                                         [MPa]  ← 파괴계수
Es = 200,000 MPa
n = Es / Ec                                               ← 탄성계수비
```

#### 4.7.2 총단면 특성

```
Ig = b × h³ / 12                                          [mm⁴]
yt = h / 2                                                [mm]
Mcr = fr × Ig / yt                                        [N·mm]  ← 균열 모멘트
```

#### 4.7.3 균열 단면 2차모멘트 Icr

```
ρ = As_bot / (b × d)
k = √(2nρ + (nρ)²) - nρ                                  ← 중립축 비율
x_cr = k × d                                              [mm]  ← 균열 중립축 깊이

Icr = b × x_cr³ / 3 + n × As_bot × (d - x_cr)²
    + (n-1) × As_top × (x_cr - d')²                       [mm⁴]  ← 압축철근 기여
```
> 압축철근(상부근)은 x_cr > d' 일 때만 기여

#### 4.7.4 서비스 모멘트

```
w_DL, w_LL: 비계수 하중 (kN/m → 처짐 계산용)
w_total = w_DL + w_LL

M_sup = w_total × L² / 12                                [N·mm]  ← 지점부
M_mid = w_total × L² / 24                                [N·mm]  ← 중앙부
```

#### 4.7.5 유효 단면2차모멘트 Ie (Branson)

> **KDS 41 20 30 4.3.3**

```
Ie(M) = { Ig                                     if M ≤ Mcr
         { (Mcr/M)³ × Ig + (1 - (Mcr/M)³) × Icr  if M > Mcr

양단 고정보 가중평균:
Ie_avg = 0.70 × Ie_mid + 0.30 × Ie_sup
```

#### 4.7.6 단기(즉시) 처짐

```
δ = w × L⁴ / (384 × Ec × Ie)                             [mm]  ← 고정-고정 등분포

δ_total_i = w_total × L⁴ / (384 × Ec × Ie_total)
δ_DL_i    = w_DL × L⁴ / (384 × Ec × Ie_DL)
δ_LL_i    = δ_total_i - δ_DL_i                            ← 활하중 즉시처짐
```

#### 4.7.7 장기처짐

> **KDS 41 20 30 4.3.4**

```
ξ = 2.0                                    ← 60개월(5년) 이상 지속하중
ρ' = As_top / (b × d)                       ← 압축 철근비
λ_Δ = ξ / (1 + 50 × ρ')                    ← 장기처짐 승수
α_sus = 0.25                                ← 활하중 지속 비율 (주거/사무)

δ_long = λ_Δ × (δ_DL_i + α_sus × δ_LL_i)   [mm]  ← 추가 장기처짐
```

#### 4.7.8 허용처짐 판정

> **KDS 41 20 30 Table 4.3-1**

```
δ_check = δ_long + δ_LL_i                  [mm]  ← 검토 총처짐

δ_allow_LL    = L / 360                      ← 활하중 즉시처짐
δ_allow_total = L / 240                      ← 비구조부재 손상 미고려
δ_allow_strict = L / 480                     ← 비구조부재 손상 고려 (참고)

판정: δ_LL_i ≤ L/360  AND  δ_check ≤ L/240 → OK
NG → h += 50mm, b 리셋 후 §4.4부터 재설계
```

### 4.8 균열 제어

> **KDS 14 20 50** — 간접균열제어 (철근 간격 제한)

```
cc = cover = 40mm
fs = (2/3) × fy                                           [MPa]
k_cr = 280 (건조환경) / 210 (기타환경)

s_limit₁ = 380 × (k_cr / fs) - 2.5 × cc                  [mm]
s_limit₂ = 300 × (k_cr / fs)                              [mm]
s_max = min(s_limit₁, s_limit₂)

실제 간격:
  edge = cover + d_stir + d_b/2
  s_actual = (b - 2×edge) / (n_bars - 1)                  [mm]  (중심~중심)

판정: s_actual ≤ s_max → OK
```

### 4.9 정착길이 / 이음길이

> **KDS 14 20 52** — 인장 이형철근 정착길이

```
① 기본정착길이:  ℓ_db = 0.9 × d_b × fy / (λ × √fck)       [mm]
   λ = 1.0 (보통콘크리트)

② 보정계수:
   α = 1.3 (상부철근) / 1.0 (하부철근)     ← 위치계수
   β = 1.0                                  ← 무도막
   γ = 0.8 (d_b ≤ 19.1mm) / 1.0 (기타)     ← 크기계수

③ 구속계수: (cb + Ktr) / d_b, Ktr=0 (간편법)
   cb = min(cover + d_b/2, spacing/2)
   (cb + Ktr)/d_b ≤ 2.5 (상한)

④ 인장 정착길이:
   ℓ_d = ℓ_db × (α × β × γ) / ((cb+Ktr)/d_b)
   ℓ_d = max(ℓ_d, 300mm)

⑤ 소요량 초과 감소: ℓ_d × (As_req / As_prov) ≥ 300mm

⑥ 겹이음길이 (Class B): ℓ_s = 1.3 × ℓ_d

⑦ 압축 정착길이:
   ℓ_dc = max(0.25 × d_b × fy / √fck, 0.043 × d_b × fy, 200mm)

⑧ 표준갈고리 정착길이:
   ℓ_dh = max(0.24 × d_b × fy / (λ × √fck), 8×d_b, 150mm)
```

→ 구조적 최소철근 (연속성 확보)
```

## 5. 기둥 설계

### 5.1 설계 흐름

```
1. 보 설계 결과에서 비계수 DL/LL 반력 분리
2. 기둥별 하중조합 생성 (§5.2)
3. 지배 조합 선택 (Pu 최대)
4. 초기 단면 결정 → 세장비 검토 수렴 루프 (§5.4)
5. 자중 수렴 (최대 2회 반복)
6. 접합부 전단 검토 (§5.8)
7. [옵션] IMF 내진 상세 (§6)
```

### 5.2 하중조합 생성

> **KDS 41 10 15** — 하중과 하중조합

보 비계수 DL/LL 분리:
```
V_DL_x = w_DL_x × L_x / 2                    ← X보 DL 전단반력
V_LL_x = w_LL_x × L_x / 2                    ← X보 LL 전단반력
M_DL_neg_x = w_DL_x × L_x² / 12              ← X보 DL 단부 모멘트
M_LL_neg_x = w_LL_x × L_x² / 12              ← X보 LL 단부 모멘트
(Y방향도 동일)
```

하중조합:

| 조합 | 축력 Pu | 모멘트 Mux | 조건 |
|------|---------|-----------|------|
| **LC1: 1.2D+1.6L** | Pu_add + 1.2(V_DL_x+V_DL_y) + 1.6(V_LL_x+V_LL_y) + 1.2P_self | Mux_add + 1.2M_DL_neg_x + 1.6M_LL_neg_x | 항상 |
| **LC0: 1.4D** | 1.4(V_DL_x+V_DL_y+P_self) + Pu_add | Mux_add + 1.4M_DL_neg_x | 항상 |
| **LC2: 1.2D+1.0E+1.0L** | 위 DL 1.2배 + LL 1.0배 + E_Pu | Mux_add + 1.2M_DL + 1.0M_LL + E_Mux | seismic_enabled |
| **LC3: 0.9D+1.0E** | 0.9(V_DL+P_self) + E_Pu | 0.9M_DL + E_Mux | seismic_enabled |

> P_self = (c_column/1000)² × (h_column/1000) × 24 [kN]
> 지배 조합: max(Pu) 기준 선택

### 5.3 초기 단면 결정

```
c_column_raw = max(b_beam_x, b_beam_y) + 100mm    ← 접합부 정착 여유
c_column = ⌈c_column_raw⌉₅₀                        ← 50mm 올림
Ag = c_column²                                      [mm²]
```

### 5.4 세장비 검토

> **KDS 41 20 40** — 세장 기둥

```
k = 1.0                                     ← 핀-핀 보수적 가정
l_u = h_column                               [mm] 비지지 길이
r = 0.3 × c_column                           [mm] 회전반경 (정방형)
λ = k × l_u / r                              ← 세장비

λ ≤ 22  → 단주 (short) — 증폭 없음
22 < λ ≤ 100 → 세장주 (slender) — δ_ns 모멘트 증폭
λ > 100 → 설계 불가 — 단면 증가 필요
```

**모멘트 증폭 (세장주)** (KDS 41 20 40):
```
Ec = 8500 × (fck + 4)^(1/3)                 [MPa]
Ig = c_column⁴ / 12                          [mm⁴]
β_d = 0.6                                    ← 지속하중 비율
EI = 0.4 × Ec × Ig / (1 + β_d)              [N·mm²]
Pc = π² × EI / (k × l_u)²                   [N]  ← 임계좌굴하중

Pu < 0.75 × Pc → 안정:
  Cm = 1.0                                   ← 균일모멘트 가정
  δ_ns = max(Cm / (1 - Pu/(0.75Pc)), 1.0)
  Mu_증폭 = δ_ns × Mu

Pu ≥ 0.75 × Pc → 불안정 → 단면 += 50mm, 재검토
```

### 5.5 주철근 설계 (P-M 상관도)

> **KDS 41 20 20 4.2** — P-M 상관도 기반 설계

#### 5.5.1 최소 편심

```
e_min = 15 + 0.03 × c_column                [mm]
e_actual = Mu × 1000 / Pu                    [mm]  (Pu > 0일 때)
e_actual < e_min → Mu_design = Pu × e_min / 1000
```

#### 5.5.2 유효깊이

```
cover = 40mm, 띠철근 D10 = 9.53mm
d_c = 40 + 9.53 + d_b/2                     ← 실제 주근 직경 기반
d_eff = c_column - d_c                       [mm]
d' = d_c                                     [mm]
```

#### 5.5.3 P-M 3점 포락선 모델

**점 A — 순수 압축 (KDS 41 20 20 4.2.1)**:
```
Po = 0.85 × fck × (Ag - As) + fy × As       [N]
φPn_max = 0.80 × 0.65 × Po                  [kN]
```
> φ = 0.65 (띠철근 기둥), 0.80 상한 (우발편심)

**점 B — 균형 파괴**:
```
c_b = 600 / (600 + fy) × d_eff              [mm]  ← 균형 중립축
a_b = β₁ × c_b
As' = As_total / 2    (대칭 배근 가정)

fs' = min(600 × (c_b - d') / c_b, fy)       ← 압축측 철근 응력
Pn_b = 0.85·fck·a_b·c + As'·fs' - As'·fy   [N]
Mn_b = 0.85·fck·a_b·c·(h/2 - a_b/2)
     + As'·fs'·(h/2 - d') + As'·fy·(d_eff - h/2)  [N·mm]

φPn_b = 0.65 × Pn_b                         [kN]
φMn_b = 0.65 × Mn_b                         [kN·m]
```

**점 C — 순수 휨 (Pn=0, 이분법 60회)**:
```
Pn=0이 되는 c_o를 c=[1e-6, c_column] 범위에서 이분법 탐색:
  fs'(c) = min(max(600·(c-d')/c, -fy), fy)
  Pn(c) = 0.85·fck·β₁·c·c_column + As/2·fs' - As/2·fy

c_o 산정 후:
  Mn_o 계산 (점B와 동일 식, c=c_o 대입)
  φMn_o = 0.85 × Mn_o                       [kN·m]  (인장지배 φ=0.85)
```

#### 5.5.4 안전성 판정

**A-B 구간** (압축지배, 직선 근사):
```
Pu ≥ φPn_b:
  M_limit = (φPn_max - Pu) × φMn_b / (φPn_max - φPn_b)
  Mu_design ≤ M_limit → 안전
```

**B-C 구간** (ε_t 기반 φ 보간, 50분할):
```
c를 c_b → 0까지 50등분:
  각 c에서 Pn(c), Mn(c) 계산
  ε_t = 0.003 × (d_eff - c) / c
  φ 보간 (KDS 41 20 20 4.2.2):
    ε_t ≥ 0.005 → φ = 0.85
    ε_t ≤ ε_y   → φ = 0.65
    그 외       → φ = 0.65 + (0.85-0.65)×(ε_t-ε_y)/(0.005-ε_y)

  Pu가 [P_cur, P_prev] 구간에 있으면 → M_limit 선형 보간
  Mu_design ≤ M_limit → 안전
```

#### 5.5.5 철근량 증가 루프

```
불안전 또는 ρ < 0.01 (최소 1%):
  n_col += 2 (2가닥 추가)
  n_col > 12 → 다음 규격 상향 (D22→D25→D29→D32), n=4 리셋
  ρ > 0.08 (최대 8%) → c_column += 50mm, 리셋

배근 적합성 (KDS 순간격):
  한 변 최대 철근 수 = ⌈n_col/4⌉ + 1
  s_min = max(40, 1.5×d_b)                  [mm]
  avail = c_column - 2×(cover+tie+d_b/2)
  req = (n_per_side-1) × (d_b + s_min)
  avail < req → 규격 상향 또는 단면 증가
```

### 5.6 이축 휨 — Bresler 역수하중법

> **Bresler Method** (Mux, Muy 모두 > 0이고 Pu > 0일 때 활성)

```
① 편심 산정:
   ex = Mux × 10⁶ / (Pu × 1000)             [mm]
   ey = Muy × 10⁶ / (Pu × 1000)             [mm]

② Pnx: X축만 편심(ex)에서 P-M 곡선상의 Pn — 이분법 60회
   c=[1e-3, 3c_column] 범위에서 e_calc = Mn(c)/Pn(c) == ex 만족하는 c 탐색

③ Pny: Y축만 편심(ey)에서 동일 방법
   → 정방형 기둥에서 Pnx == Pny 일 수 있음

④ Pno = 0.85·fck·(Ag - As) + fy·As          [N]  ← 순수 압축

⑤ Bresler:
   1/Pn = 1/Pnx + 1/Pny - 1/Pno
   Pn_bresler = 1 / (1/Pnx + 1/Pny - 1/Pno)

⑥ φPn = 0.65 × Pn_bresler                   [kN]
   판정: Pu ≤ φPn → OK
```

> SRSS 값 (√(Mux²+Muy²))은 단축 P-M 검토용 비교 참고값으로 유지

### 5.7 띠철근 설계

> **KDS 41 20 20**

```
주근 ≤ D32 → 띠철근 D10 (9.53mm)
주근 ≥ D35 → 띠철근 D13 (12.7mm)

최대 간격 = min(
  16 × 주근 직경,
  48 × 띠철근 직경,
  c_column                                    ← 단면 최소 치수
)
s_tie = ⌊max_spacing / 50⌋ × 50             ← 50mm 내림
```

### 5.8 접합부 전단 검토

> **KDS 41 17** — 보-기둥 접합부

```
α_o = 1.25 (IMF) / 1.0 (OMF)                ← 응력 증폭

T = As_beam_top × α_o × fy / 1000           [kN]  ← 보 상부근 인장력
V_col = 2 × |M_neg| / (h_column/1000)       [kN]  ← 기둥 전단력 (대칭)
Vj = T - V_col                               [kN]  ← 접합부 전단력

Aj = c_column × min(c_column, b_beam + c_column)  [mm²]  ← 유효면적

γ 계수:
  내부접합/IMF: 1.7    내부접합/OMF: 1.0
  외부접합/IMF: 1.25   외부접합/OMF: 0.75

φVn = 0.75 × γ × √fck × Aj / 1000          [kN]
판정: |Vj| ≤ φVn → OK
```

### 5.9 자중 수렴 루프

```
단면이 커지면 자중(P_self)이 변하므로 하중조합 재생성:
  P_self = (c_column/1000)² × (h_column/1000) × 24  [kN]

1차 설계 → c_final 결정 → P_self 재산정 → 하중조합 재생성 → Pu 차이 < 0.5kN → 수렴
최대 2회 반복
```

### 6.1 보 소성힌지 구간 (KDS 41 17 4.4)

### 6.2 기둥 구속구간 (KDS 41 17 4.5)

### 6.3 강기둥-약보 (KDS 41 17 4.5)

```
Mn_col: 현재 배근+축력(Pu)에서의 기둥 공칭 휨강도
  → c를 이분법 60회로 Pu에 대응하는 Mn 역산

ΣMn_col = 2 × Mn_col                        ← 상하 기둥 (대칭)
ΣMn_beam = Mn_beam_x + Mn_beam_y            ← 양측 보

보 공칭 휨강도: Mn = As × fy × (d - a/2) / 10⁶  [kN·m]
  a = As × fy / (0.85 × fck × b)

판정: ΣMn_col / (1.2 × ΣMn_beam) ≥ 1.0 → OK
```

## 10. 주요 가정 및 제한 사항

1. **구조 모델**: 양단 고정보 (등분포하중), 정방형 기둥, 대칭 배근
2. **부재력 모델**: M_neg = wL²/12, M_pos = wL²/24, V_max = wL/2
3. **세장비**: 횡구속 프레임 가정 (k=1.0, 보수적 핀-핀)
4. **기둥 P-M**: 대칭 배근 (As' = As = As_total/2), 3점 포락선 + B-C 50분할
5. **Bresler**: 정방형 기둥에서 X/Y 대칭이므로 Pnx ≈ Pny
6. **처짐**: Branson Ie, 양단고정 가중평균 (0.70·Ie_mid + 0.30·Ie_sup)
7. **장기처짐**: ξ=2.0 (5년), α_sus=0.25 (주거/사무)
8. **콘크리트**: 보통중량 (γ=24 kN/m³, λ=1.0)
9. **접합부**: 내부접합부 가정 (기본값), 양쪽 보 대칭
10. **강기둥-약보**: 상하 기둥 대칭 (ΣMn_col = 2×Mn_col)

## 11. 검토 모드 계산 로직 (`review/calculation_review.py`)

> v0.7.0에서 추가. 구조계산서(MIDAS/BeST) 부재력을 직접 입력받아 OK/NG 판정.
> 기존 설계 엔진과 **완전 독립** (방식B — KDS 수식 별도 구현).

### 11.1 설계 모드와의 차이

| | 설계 모드 | 검토 모드 |
|---|---|---|
| 입력 | 면하중 (kN/m²) | 부재력 (Mu, Vu, Pu) 직접 |
| 부재력 산정 | wL²/12 등 자동 계산 | MIDAS가 이미 산출 |
| 배근 결정 | As_req → rebar 탐색 → 자동 | 구조계산서 배근 그대로 사용 |
| φMn 계산 | As_provided(자동 탐색) 기반 | As_provided(구조계산서 배근) 기반 |
| 수렴루프 | 있음 (h/b 자동 증가) | 없음 (단면 고정) |
| 유효깊이 d | cover + stirrup + rebar/2 | **Loc 기반** (d = h - Loc) |

### 11.2 보 검토 흐름

```
입력: Mu_neg, Mu_pos, Vu (각 END-I/MID/END-J), h, b, fck, fy, Loc, 배근
  ↓
1. d = h - Loc (Loc 기반)
2. 구조계산서 배근 파싱: "3-D19" → As = 859.5 mm²
3. φMn 계산: a = As·fy/(0.85·fck·b), c = a/β₁, εt → φ → φMn
4. Check Ratio = Mu / φMn (>1.0이면 NG)
5. 전단: φVc + φVs vs Vu
```

### 11.3 기둥 검토 흐름

```
입력: Pu, Mux, Muy, bx, by, h_column, fck, fy
  ↓
1. 세장비 λ = k·lu/r, δ_ns 모멘트 확대
2. P-M 3점(A/B/C) + 50분할 포락선
3. Bresler 쌍축휨: 1/Pn = 1/Pnx + 1/Pny - 1/Pno
4. φPn vs Pu → OK/NG
```

### 11.4 3D 배근도 어댑터

검토 결과를 `plot_3d_frame_rebar` 호환 형식으로 변환:
- 천장보 X/Y + 바닥보 X/Y + 기둥 매핑
- 배근 문자열 → rebar_steps dict 생성
- 바닥보 "없음" → 천장보 폴백
